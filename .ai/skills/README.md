# Skills — Compétences IA

Ce dossier contient les compétences custom réutilisables par les agents IA.

## Format d'un skill

Chaque skill est un fichier markdown nommé `<nom>.md` avec la structure suivante :

```markdown
# <Nom du skill>

## Contexte d'usage
Décrire quand utiliser ce skill.

## Prompt type
Le prompt ou workflow à appliquer.

## Critères de succès
Comment valider que le skill a bien été exécuté.
```
