# Plugins — Configuration des extensions et MCP servers

Ce dossier contient la configuration des extensions et MCP servers utilisés par les agents IA.

## Format

Chaque configuration est un fichier JSON ou markdown nommé `<plugin>.md` (ou `<plugin>.json`) décrivant :
- Le nom du plugin/MCP server
- Sa configuration
- Son usage
