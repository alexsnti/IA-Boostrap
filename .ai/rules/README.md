# Rules — Règles et conventions

Ce dossier contient les règles métier et conventions de code à respecter par les agents IA.

## Format d'une règle

Chaque règle est un fichier markdown nommé `<domaine>.md` avec la structure suivante :

```markdown
# <Domaine>

## Règle
Énoncer la règle clairement.

## Justification
Pourquoi cette règle existe.

## Exemples
Exemples de respect et de violation de la règle.
```
